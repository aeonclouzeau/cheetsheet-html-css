Web Fonts

A web font is any font used in a website’s design that isn’t installed by default on the end user’s device—a counterpart to a system font.

Online font services, like Google Fonts, make it easy to find and link to fonts from your site.

Link

When selecting a font and its styles, you can review your selected font family, and a <link> element will be automatically generated.

Apply web font via link
<head> <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@100&display=swap" rel="stylesheet" /></head>

The generated <link /> element needs to be added to the <head> element in the HTML document for it to be ready to be used in the CSS stylesheet.


p { font-family: 'Roboto', sans-serif; }

Syntax

@import

@font-face

Add a web font to the document with the @font-face property and the path to the font's source.

Use of @font-face rulset
@font-face { font-family: 'MyParagraphFont'; src: url('fonts/Roboto.woff2') format('woff2'), url('fonts/Roboto.woff') format('woff'), url('fonts/Roboto.ttf') format('truetype'); }

Once the @font-face ruleset is defined, you can use the font in your stylesheet!

p { font-family: 'MyParagraphFont', sans-serif; }

Font formats

Fonts can be downloaded just like any other file on the web. They come in a few different file formats, such as:

  • OTF (OpenType Font)
  • TTF (TrueType Font)
  • WOFF (Web Open Font Format)
  • WOFF2 (Web Open Font Format 2)
Note

The different formats are a progression of standards for how fonts will work with different browsers, with WOFF2 being the most progressive. It’s a good idea to include TTF, WOFF, and WOFF2 formats with your @font-face rule to ensure compatibility on all browsers.
